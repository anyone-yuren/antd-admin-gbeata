# 画板组件

> 基于Konva和react-konva编写的简单画板组件，鼠标移上去即可当作画笔

安装`pnpm add ui --filter @gbeata/admin-docs`
<code src="./demos/drawboard"></code>

## API

| 属性名    | 描述         | 类型            | 默认值 |
| --------- | ------------ | --------------- | ------ |
| stroke  | 画笔颜色       | string | `'#df4b26'` |
| strokeWidth | 画笔大小     | number | 5 |
